# Rust

## Chapter 0 - Getting Getting Started

Peguei o setup inicial para desenvolver `algo` em Rust.

## Chapter 1 - True Getting Started

Aprendi os tipos primitivos da linguagem ali, básico para todo código. Aproveitei do conhecimento para modular melhor os testes, mas fingirei que não usei conhecimento mais avançados.

Tipos mais complexos estudados como Arrays, Tuples e entendi melhor a lógica de Referências e Fat Pointers em Rust, além de ter dado uma olhada melhor em coisas como heap e stack.

## Chapter 2 - Programming a Guessing Game

Basicamente aprendi como funciona a importação de crates em Rust, como funciona o retorno de algumas funções como Ok e Err, estruturas como match e interações de i/o usando a crate nativa.
